    Особенности проекта

    Для проекта был создан аргономичный дизайн с удобной для пользователей навигацией для поиска необходимого товара и полезной информации.

    Калькулятор

    Это уникальный инструмент расчета скидки: всплывающие подсказки при выборе значений, влияющих на стоимость. Под каждую группу товаров создан уникальный калькулятор с учетом особенностей продукта.

    Процент маржи рассчитывается исходя из общего объема заказа, чтобы посчитать итоговую сумму без участия менеджера. Интеллектуальный подбор товара: основан на технических параметрах, а не маркетинговых характеристиках.

    Интеграция с AmoCRM: заявка, поступившая с сайта, движется в соответствии с бизнес-процессами компании.
